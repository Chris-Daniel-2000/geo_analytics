Analysis Procedures

This page outlines the steps involved in analyzing the uploaded Excel file in the Skip Trace Analytics application.

Steps for Analysis

  1. Upload File: The user uploads an Excel file containing geographical coordinates.
  2. Data Processing: The application processes the uploaded file to extract and analyze the coordinates.
  3. Analysis Method: The user selects an analysis method (e.g., clustering, heatmap generation).
  4. Results Generation: The application generates results based on the selected analysis method.
  5. Download Processed File: The user can download the processed file containing the analysis results.
  6. Review Results: Paste the most frequent cordinates generate by the pivot table on the google maps to validate see the satelite location, ensure that you choose location which are not on the road

For more information, please refer to the documentation or contact support.
